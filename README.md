# Stratify
Stratify - Sistema Automatizado de Gestión de InventariosStratify es una solución tecnológica diseñada para optimizar la gestión, el control y el flujo de inventarios en tiempo real para pequeñas y medianas empresas (PYMES). Bajo el marco de metodologías ágiles (SCRUM) y fundamentado en normativas de modernización empresarial , este sistema mitiga los errores humanos y reduce las pérdidas económicas ocasionadas por la falta de automatización y el vencimiento de productos.  🚀 Características PrincipalesControl en Tiempo Real: Registro preciso de las entradas y salidas de mercancía, asociando fecha, cantidad y el usuario responsable.  Alertas Inteligentes: Notificaciones automáticas cuando un producto se encuentra fuera de stock o está próximo a alcanzar su fecha de vencimiento.  Gestión de Stock Eficiente: Validación y configuración de límites de almacenamiento máximo y mínimo por producto para evitar sobrecostos o desabastecimiento.  Seguridad y Roles: Autenticación de usuarios y asignación de roles específicos con permisos configurables para proteger la integridad de los datos.  Reportes y Analítica: Generación de informes mensuales de costos, existencias y métricas sobre los productos más o menos vendidos.  🛠️ Arquitectura y Diseño del SistemaEl proyecto cuenta con un diseño de software estructurado que incluye:Casos de Uso: Modelado del comportamiento del sistema y la interacción de los actores (Administrador, Cliente y Sistema).  Diagrama de Clases: Arquitectura orientada a objetos que define las entidades principales como Producto, Inventario, Usuario y Factura.  Modelo Entidad-Relación (MER) y Lógico: Estructura de base de datos relacional normalizada para garantizar la persistencia e integridad de la información.
